# DataLens - 智慧型資料表格閱讀器

DataLens 是一個輕量且功能強大的網頁工具，專為快速檢視與分析 CSV 和 JSON 資料而設計。無需安裝任何軟體，直接在瀏覽器中開啟即可使用。

## ✨ 主要功能

*   **多格式支援**：支援 CSV、JSON 與 TXT 檔案格式。
*   **靈活輸入**：可直接拖放檔案上傳，或貼上純文字內容。
*   **智慧解析**：
    *   自動偵測 JSON 結構，若包含多個陣列資料，可透過下拉選單切換檢視不同資料表。
    *   CSV 支援自訂編碼（UTF-8、Big5）與首行標題設定。
*   **互動式表格**：
    *   **排序**：點擊欄位標題即可進行升冪或降冪排序。
    *   **搜尋**：即時全文檢索，快速篩選資料。
    *   **複製**：點擊任意儲存格或行號，即可將內容複製到剪貼簿。
*   **使用者體驗優化**：
    *   支援固定首欄與標題列，方便瀏覽大量資料。
    *   Session Storage 自動暫存，重新整理頁面不丟失資料。

## 🚀 如何使用

1.  下載本專案所有檔案。
2.  使用瀏覽器（建議 Chrome、Edge 或 Safari）開啟 `index.html`。
3.  **上傳資料**：
    *   點擊上傳區塊選擇檔案。
    *   或將檔案拖曳至視窗中。
    *   或是直接將文字貼上至輸入框並點擊「轉換」。
4.  **開始分析**：使用上方工具列進行搜尋，或點擊表頭進行排序。
